Project 2 - CrewBot (Discord API bot)

This project consisted of a web server utilizing Express via Node.js, along with using Discord.js to interact with Discord's API. It used information fed to it about a given round of Among Us to automatically handle muting and deafening players inside of Discord, as well as adding extra game depth by changing how some players were deafened. It used JavaScript for the server, and primarily JS for the front end with some HTML. The original version used Processing.js to render everything, but the more recent versions instead relied more on heavily-styled HTML instead. The project used WebSockets for communication between the client and server.
